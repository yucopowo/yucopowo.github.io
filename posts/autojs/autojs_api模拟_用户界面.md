# autojs_api模拟_用户界面

## 用户界面: UI


```autojs {demo}
"ui";

ui.layout(
    <horizontal>
        <button id="ok" text="第一个按钮"/>
        <button text="第二个按钮"/>
    </horizontal>
);

```

## 帧布局: frame


```autojs {demo}
"ui";

ui.layout(
    <frame>
        <button id="ok" text="第一个按钮"/>
        <button text="第二个按钮"/>
    </frame>
);

```
